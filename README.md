# Getting Started
## 测试构架

本项目模拟了一个电影网站的存储场景，主要功能为上传电影和给电影评分。测试人员通过为`MovieRepository`和`UserRepository`两个接口提供不同数据库的实现，来构建不同数据库的应用实例，从而测试数据库在真实应用场景下的性能。
